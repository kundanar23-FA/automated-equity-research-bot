\### \*\*Project Overview\*\*

This is an automated financial analysis tool built with Python. It scans the \*\*Nifty 50\*\* stocks in real-time, fetches fundamental data (Book Value, EPS, P/E), and calculates the \*\*Intrinsic Value\*\* using Benjamin Graham's formula.



It solves the problem of manual data entry for Equity Research Analysts by automating the valuation process.



\### \*\*Features\*\*

\* \*\*Algorithmic Valuation:\*\* Automatically calculates `Sqrt(22.5 \* EPS \* Book Value)` to find fair price.

\* \*\*Risk Analysis:\*\* Flags stocks with "High Debt" or negative margins.

\* \*\*Live Dashboard:\*\* Generates an HTML Visual Dashboard with Buy/Sell signals.

\* \*\*Excel Export:\*\* Creates a CSV report for further analysis.



\### \*\*Tech Stack\*\*

\* \*\*Python 3.10+\*\*

\* \*\*yFinance API\*\* (Real-time Market Data)

\* \*\*Pandas\*\* (Data Manipulation)
